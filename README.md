# Cadastro de usuários
Projeto de implementação de cadastro de usuários, usando as seguintes bibliotecas:
- NestJS;
- JWT;
- Mongoose;
## Instalação
```npm i```
## Uso

```http://localhost:8080/cad-user/health```
```http://localhost:8080/cad-user/cadastro/search/:id```

```npm run pretest -- --fix```
